<!DOCTYPE html>

DOCTYPE html tag tells the interpreter (in most cases : Browsers) that given file format is of Html5 or written in Html5. It is helpful for browser because there are different attributes and behaviors associated with Html5 only which are not available in previous versions of Html5 (example: header, footer and many other tags).

Demonstration Example :
<!DOCTYPE html> <html lang="en"><!--Tells The Language Of Content In Html Page--> </html>
