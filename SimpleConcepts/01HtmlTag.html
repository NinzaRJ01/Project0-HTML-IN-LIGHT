<!DOCTYPE html>

DOCTYPE html tag tells the compiler (in most cases : Browsers) that given file format is of Html5 or written in Html5. It is helpful for browser because there are different attributes and methods associated with Html5 only which are not avaiable in previous versions of Html5 (example: header, footer and many other tags).
